KeyChain is a cyber security integration software for protecting private keys in any signature-dependent system. It has a user interface for signing transactions, generating and managing hierarchically-derived keys. KeyChain supports Bitcoin and Ethereum transactions, expanding fast to include Litecoin, Ethereum Classic, Bitcoin Cash, and Bitshares.

OS-specific security

We look past standard solutions to data protection. As is well-known, the longer the path, the more chances there are of passwords being intercepted. To ensure safe operations in Unix-like OS, we have created a shortcut from you entering a passphrase to the KeyChain processing it. To protect Windows users, we employ the mechanism that allows the processes to run separately without letting any one of them to interfere with another - a technology that is inherent to macOS.

API layer

A language neutral layer which integrates with your app or website. It transmits and parses commands for a given API through pipes and websocket. Learn more

Security layer

KeyChain’s ace in the sleeve! It provides the safest communication channel between the user and Keychain. It is an OS specific structure and interacts with the system itself without any external exposure. Learn more

Signing module

A mezo-structure which interacts with the Security layer and unlocks the keys once the correct passphrase has been given to KeyChain. Learn more
